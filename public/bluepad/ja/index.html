# BluePad
 
## はじめに
 
BluePadへようこそ。
新しいマークダウンエディタです。速く、きれいで、
シンプルさを大切にする方のために。
 
```javascript
const editor = new BluePad();
editor.write("Hello, world!");
```
Features

必要なもの全て、
余計なものゼロ。

速さと明快さのために設計。肥大化なし、Electronなし — Tauri搭載の高速ネイティブアプリ。

✎

WYSIWYG + ソースモード

ショートカット一つでリッチテキスト編集とマークダウンソースを自在に切替。リアルタイムフォーマット確認または直接構文編集が可能です。

📄

マルチタブ編集

クリーンなタブインターフェースで複数ドキュメントを同時作業。進行状況を失わずに素早くコンテキスト切替できます。

🎨

4つの美しいテーマ

クラシックライト、ソフトセピア、ダークモード、ディープナイト。気分と環境に合ったパレットを選んでください。

🕶

フォーカスモード

現在の段落以外をすべて薄くします。気が散る要素を排除し、ライティングフローを維持します。

🔍

検索 & 置換

正規表現対応の強力な検索機能。ドキュメント全体でテキストを見つけて即座に置換できます。

🌐

多言語対応

英語、韓国語、日本語UIを完全サポート。適切なCJKテキストレンダリングで、あらゆる言語で執筆できます。
